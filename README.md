## Recommended pre-processing of input waveforms for PKIKP onset picking
- Format: .sac
- Channel: Vertical channel
- Length: 50 s around ak135 prediction
- Sampling rate: 40 Hz
- Frequency filtering: 0.5 - 2 Hz
